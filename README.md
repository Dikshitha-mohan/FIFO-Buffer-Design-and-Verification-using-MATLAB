# FIFO Buffer Design and Verification using MATLAB

## Overview
This project implements a First-In First-Out (FIFO) buffer in MATLAB.

## Features
- FIFO memory implementation
- Write operation
- Read operation
- Full flag detection
- Empty flag detection
- Data verification

## Technologies Used
- MATLAB

## Results
The FIFO successfully stores and retrieves data while maintaining the correct order.
